## Installation

This project is bootstrapped by [Create React App][cra] so read their doc if you don't know what to do.

Clone this repository and change to directory

```sh
cd jinhii-site
```

Install dependencies

```sh
npm install -d
```

Start local development server

```sh
npm start
```
